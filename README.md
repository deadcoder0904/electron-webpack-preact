# electron-webpack-preact

## Installation

### 1. Clone the repo

```bash
git clone git@github.com:deadcoder0904/electron-webpack-preact.git
```

### 2. Install the packages

```bash
npm install
```

### 3. Watch `webpack` in one terminal

```bash
npm run build:watch
```

### 4. Launch `electron` in another terminal

```bash
npm start
```
